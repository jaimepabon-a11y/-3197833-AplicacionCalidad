
# 3197833-AplicacionCalidad – Sistema GYM

Aplicación web desarrollada para la gestión de productos, categorías y procesos internos de un gimnasio.  
Permite realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar), manejar sesiones de usuario, validar datos y administrar inventario de manera sencilla.


##  Descripción del Proyecto


El sistema GYM es una aplicación que permite:

-   Gestionar productos del gimnasio (nombre, precio, stock, categoría).
    
-   Administrar categorías disponibles.
    
-   Controlar el acceso mediante inicio y cierre de sesión.
    
-   Aplicar validaciones para evitar datos incompletos o inválidos.
    
-   Realizar operaciones CRUD usando PHP y MySQL.
    
-   Mantener una estructura modular para facilitar mantenimiento y escalabilidad.
    

El propósito del proyecto es demostrar buenas prácticas en calidad de software y administración de inventarios en un entorno de gimnasio.


## Tecnologías Utilizadas
PHP– Lenguaje principal del sistema.
    
MySQL– Base de datos.
